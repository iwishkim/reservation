# 🍽️ 식당 예약 시스템

웹 기반 식당 예약 애플리케이션입니다. 사용자가 쉽게 식당 예약을 하고 관리할 수 있습니다.

## ✨ 주요 기능

- **예약 신청**: 이름, 전화번호, 날짜, 시간, 인원, 식당 선택
- **특별 요청사항**: 추가 요청사항 입력 (예: 창가 자리, 생일 파티 등)
- **예약 목록**: 예약한 내용을 카드 형태로 표시
- **예약 관리**: 예약 삭제 기능
- **로컬 스토리지**: 브라우저에 데이터 자동 저장

## 📁 프로젝트 구조

```
myapp/
```

## 🚀 사용 방법

### 1. 파일 열기
브라우저에서 `index.html` 파일을 열어줍니다.

### 2. 예약 신청
1. **이름** 입력
2. **전화번호** 입력
3. **예약 날짜** 선택 (오늘 이후만 가능)
4. **예약 시간** 선택
5. **인원 수** 선택 (1~8명)
6. **식당** 선택
7. **특별 요청사항** 입력 (선택사항)
8. **예약 신청** 버튼 클릭

### 3. 예약 관리
- **예약 현황** 섹션에서 모든 예약을 확인
- 예약 카드의 **삭제** 버튼으로 예약 취소
- 예약은 자동으로 날짜 순서대로 정렬됨

## 🎨 주요 기능 설명

### 예약 폼
- 모든 필수 항목(*) 입력 필요
- 날짜는 오늘부터만 선택 가능
- 실시간 유효성 검사

### 예약 카드
- **식당명**: 예약한 식당 이름
- **상태**: 예약 완료 표시
- **예약자 정보**: 이름, 전화번호
- **예약 일시**: 날짜와 시간 (한국식 형식)
- **인원 수**: 예약 인원
- **특별 요청**: 입력한 추가 요청사항

### 알림 메시지
- ✅ **성공**: 예약 완료 시 표시
- ❌ **오류**: 유효하지 않은 입력 시 표시
- ℹ️ **정보**: 예약 삭제 시 표시

## 💾 데이터 저장

예약 정보는 브라우저의 로컬 스토리지에 자동 저장되므로:
- 페이지를 새로고침해도 예약 정보가 유지됨
- 브라우저 캐시를 지우면 데이터가 삭제됨
- 각 브라우저에서만 데이터가 저장됨

## 🎯 사용 예시

1. 예약자: 김철수
2. 전화: 010-1234-5678
3. 날짜: 2026년 6월 15일
4. 시간: 19:00
5. 인원: 4명
6. 식당: 이탤리언 플레이스
7. 요청: 창가 자리 부탁드립니다

→ 예약 신청 버튼 클릭 → 완료!
